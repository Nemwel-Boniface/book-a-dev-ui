## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- node js, npm

### Setup

- clone repo `git clone https://github.com/maxthestranger/book-a-dev-ui`
- cd into repo `cd book-a-dev-ui`

### Install

- run `npm install`

### Start

- run `npm start`

### Usage

- `http://localhost:3000`
